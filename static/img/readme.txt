This folder contains image files
used for the quarto app.

The file name is piece-specific. Do not change.
Please convert that unique number into binary.
You will know about the rules below.
<bin rules>
bit-0: Color  => 0: White   1: Black
bit-1: Shape  => 0: Round   1: Square
bit-2: Height => 0: Low     1: High
bit-3: Hole   => 0: No hole 1: With hole

png size: 120px * 120px
